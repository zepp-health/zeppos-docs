在本章节中，我们将完成以下环境准备工作。

- 安装 Node.js（推荐 LTS 版本）
- 安装 Zeus CLI

:::info

如果在安装过程中遇到问题，建议将报错信息、操作系统版本和当前执行的命令发给 AI 进行排查，通常能更快解决问题。
:::

## 安装 Node.js

访问 [Node.js 官网 - 下载 Node.js](https://nodejs.org/zh-cn/download/package-manager)。

推荐使用官方安装程序进行安装，安装 LTS 版本。

如果习惯使用 Node.js 版本管理工具，建议参考官方的命令安装指引。

以下安装命令摘自官网，是在 macOS 上使用 fnm 安装 Node.js 的示例，其他平台的安装方式请参考官网指引。

```sh
# 下载并安装 fnm：
curl -o- https://fnm.vercel.app/install | bash

# 下载并安装 Node.js：
fnm install 24

# 验证 Node.js 版本：
node -v # Should print "v24.15.0".

# 验证 npm 版本：
npm -v # Should print "11.12.1".
```

:::info

- 在 Windows 中推荐使用系统内置的 `PowerShell` 终端来执行命令，以管理员方式运行
- 在 MacOS 和 Linux 中推荐使用系统内置的 `Terminal` 终端来执行命令

:::

## 安装 Zeus CLI

[Zeus CLI](../tools/cli/index.md) 是一个 Zepp OS 命令行工具集，我们用它来创建、预览编译以及构建 Zepp OS 小程序或者表盘项目。

第一步，在终端执行安装命令。

```sh
npm i @zeppos/zeus-cli -g
```

:::tip
安装 Zeus CLI 过程中遇到问题请查看 [环境安装常见问题](../faq/env-setup.md)
:::

第二步，检查 Zeus CLI 是否安装成功，在终端执行以下命令，如果终端显示如 `1.9.0` 的版本号，则表明安装成功。

```sh
zeus -v
```
