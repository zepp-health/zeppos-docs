# 模拟器下载

Zepp OS 模拟器全新推出 v2 版本，大大简化流程，无需虚拟网卡等复杂配置，安装与启动非常方便。

根据系统版本，下载对应的模拟器安装包，根据提示进行安装。

安装完成后运行模拟器，点击模拟器图标，即可运行。

以 Windows 举例，双击模拟器图标即可启动模拟器。
