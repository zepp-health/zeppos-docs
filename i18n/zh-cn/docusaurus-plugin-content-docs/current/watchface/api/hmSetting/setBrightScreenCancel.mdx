取消亮屏时间。

调用过 `setBrightScreen` 方法设置过亮屏时间后，需要在 app 销毁函数中调用 `setBrightScreenCancel` 方法。

## 类型

```ts
() => result
```

## 参数

### result

| 说明                       | 类型     |
| -------------------------- | -------- |
| 操作结果，`0` 表示设置成功 | `number` |

## 代码示例

```js
hmSetting.setBrightScreen(10)
hmSetting.setBrightScreenCancel()
```
