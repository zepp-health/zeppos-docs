获取用户体重目标。

## 类型

```ts
() => weightTarget
```

## 参数

### weightTarget

| 说明                                     | 类型     |
| ---------------------------------------- | -------- |
| 用户设置的体重目标，默认为 `0`，单位千克 | `number` |

## 代码示例

```js
const target = hmSetting.getWeightTarget()
```
