返回当前设备的屏幕亮度。

## 类型

```ts
() => result
```

## 参数

### result

| 说明                              | 类型     |
| --------------------------------- | -------- |
| 当前设备的屏幕亮度，范围 [0, 100] | `number` |

## 代码示例

```js
const result = hmSetting.getBrightness()
```
