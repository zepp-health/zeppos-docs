返回当前的语言序号。

## 类型

```ts
() => result
```

## 参数

### result

| 说明                       | 类型     |
| -------------------------- | -------- |
| 语言序号，与国家对应，参考 [多语言映射](../../../reference/related-resources/language-list.mdx) | `number` |

## 代码示例

```js
const language = hmSetting.getLanguage()
```
