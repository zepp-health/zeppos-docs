# ECDSA

## createCrypto

创建 ECDSA 数字签名实例，默认使用 `ecp_dp.SECP256K1`。

| 参数值             | 说明                       | API_LEVEL |
| ------------------ | -------------------------- | --------- |
| `alg.ECDSA`        | ECDSA 数字签名             | `3.0`     |
| `ecp_dp.SECP192K1` | SECP192K1 椭圆曲线         | `3.0`     |
| `ecp_dp.SECP224K1` | SECP224K1 椭圆曲线         | `3.0`     |
| `ecp_dp.SECP256K1` | SECP256K1 椭圆曲线，默认值 | `3.0`     |

### 类型

```ts
function createCrypto(algorithmId: typeof alg.ECDSA, option?: ECDSAOptions): ECDSACrypto | undefined
```

### 参数

#### ECDSAData

| 类型                                                   | 说明         |
| ------------------------------------------------------ | ------------ |
| <code>number[]&#124;ArrayBuffer&#124;Uint8Array</code> | 待处理的数据 |

#### ECDSAOptions

| 属性        | 类型                                                                                           | 必填 | 默认值 | 说明                                  |
| ----------- | ---------------------------------------------------------------------------------------------- | ---- | ------ | ------------------------------------- |
| ecp_dp_mode | <code>typeof ecp_dp.SECP192K1&#124;typeof ecp_dp.SECP224K1&#124;typeof ecp_dp.SECP256K1</code> | 否   | -      | 椭圆曲线参数，默认 `ecp_dp.SECP256K1` |
| key_encrypt | <code>boolean&#124;number</code>                                                               | 否   | -      | 密钥是否经过硬件密钥二次加密          |
| private_key | <code>ECDSAData</code>                                                                         | 否   | -      | 已有私钥                              |
| pub_key     | <code>ECDSAData</code>                                                                         | 否   | -      | 已有公钥                              |

## ECDSACrypto

ECDSA 实例，用于创建密钥对、生成签名和验证签名。

### 方法

#### createChiper

创建 ECDSA 公钥和私钥；失败时返回 `undefined`

```ts
createChiper(): ECDSAKeyResult | undefined
```

##### ECDSAKeyResult

| 属性               | 类型                     | 说明     |
| ------------------ | ------------------------ | -------- |
| private_key_length | <code>number</code>      | 私钥长度 |
| private_key        | <code>ArrayBuffer</code> | 私钥数据 |
| pub_key_length     | <code>number</code>      | 公钥长度 |
| pub_key            | <code>ArrayBuffer</code> | 公钥数据 |

#### encrypt

为数据生成数字签名

```ts
encrypt(
      data: createCrypto.ECDSAData,
      option: Options,
    ): ECDSACipherResult | undefined
```

##### Options

| 属性        | 类型                                                | 必填 | 默认值 | 说明                                      |
| ----------- | --------------------------------------------------- | ---- | ------ | ----------------------------------------- |
| digest_type | <code>typeof alg.MD5&#124;typeof alg.HMACMD5</code> | 是   | -      | 摘要算法，支持 `alg.MD5` 和 `alg.HMACMD5` |

##### ECDSACipherResult

| 属性   | 类型                     | 说明         |
| ------ | ------------------------ | ------------ |
| data   | <code>ArrayBuffer</code> | 数字签名数据 |
| length | <code>number</code>      | 数据长度     |

#### decrypt

使用签名数据验证原始数据

```ts
decrypt(
      data: createCrypto.ECDSAData,
      option: Options,
    ): ECDSACipherResult | undefined
```

##### Options

| 属性     | 类型                                | 必填 | 默认值 | 说明             |
| -------- | ----------------------------------- | ---- | ------ | ---------------- |
| sig_data | <code>createCrypto.ECDSAData</code> | 是   | -      | 待验证的数字签名 |

##### ECDSACipherResult

| 属性   | 类型                     | 说明             |
| ------ | ------------------------ | ---------------- |
| data   | <code>ArrayBuffer</code> | 签名验证结果数据 |
| length | <code>number</code>      | 数据长度         |

### 代码示例

```js
import { alg, createCrypto, ecp_dp } from '@zos/crypto'

const source = new Uint8Array([1, 2, 3])
const ecdsa = createCrypto(alg.ECDSA, { ecp_dp_mode: ecp_dp.SECP256K1 })
if (!ecdsa) throw new Error('Failed to create ECDSA instance')

const keys = ecdsa.createChiper()
const signature = ecdsa.encrypt(source, { digest_type: alg.MD5 })
if (signature) {
  const verified = ecdsa.decrypt(source, { sig_data: signature.data })
}
```
