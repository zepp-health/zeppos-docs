获取临时存储的 64 位整数，系统重启将清除

## 类型

```ts
(key: string) => result
```

## 参数

### key

| 说明     | 必填 | 类型     | 默认值 |
| -------- | ---- | -------- | ------ |
| 键字符串 | 是   | `string` | -      |

### result

| 说明             | 类型     |
| ---------------- | -------- |
| 存储的 64 位整数 | `number` |

## 用法

```js
hmFS.SysProSetInt64('js_test_int64', 200)
console.log(hmFS.SysProGetInt64('js_test_int64'))
```
