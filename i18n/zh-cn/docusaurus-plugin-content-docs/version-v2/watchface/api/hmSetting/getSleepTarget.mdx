获取用户睡眠目标。

## 类型

```ts
() => sleepTarget
```

## 参数

### sleepTarget

| 说明                                      | 类型     |
| ----------------------------------------- | -------- |
| 用户设置的睡眠目标，默认为 `0`，单位为分钟 | `number` |

## 代码示例

```js
const target = hmSetting.getSleepTarget()
```
