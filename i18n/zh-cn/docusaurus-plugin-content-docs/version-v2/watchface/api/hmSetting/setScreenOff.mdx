title: hmSetting.setScreenOff()
sidebar_label: setScreenOff

调用 `setScreenOff` 即可息屏。

## 类型

```ts
() => result
```

## 参数

### result

| 说明                       | 类型     |
| -------------------------- | -------- |
| 操作结果，`0` 表示设置成功 | `number` |

## 代码示例

```js
const result = hmSetting.setScreenOff()
```
