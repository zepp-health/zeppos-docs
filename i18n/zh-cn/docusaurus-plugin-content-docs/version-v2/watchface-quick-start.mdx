## 创建第一款表盘

我们提供了两种构建表盘的方式。

第一种在浏览器中通过 [表盘制作工具](https://watchface.zepp.com/)（可视化交互界面）在线制作表盘。

![effects](/img/docs/guides/tools/watchface/overview.png)

:::info
表盘制作工具无需任何编程经验，操作和交互设计的非常友好！更多用法参考 [表盘制作工具 - 使用说明](../guides/tools/watchface/index.md)
:::

第二种是通过 Zeus CLI 的方式创建表盘项目，与开发小程序的方式相同，请参考 [快速上手](../guides/quick-start.mdx)。
