Returns the current language serial number.

## Type

```ts
() => result
```

## Parameters

### result

| Description                                                                                                                       | Type     |
| --------------------------------------------------------------------------------------------------------------------------------- | -------- |
| Language serial number, corresponding to country, reference [Multilingual Mapping](../../related-resources/language-list.mdx) | `number` |

## Code example

```js
const language = hmSetting.getLanguage()
```
