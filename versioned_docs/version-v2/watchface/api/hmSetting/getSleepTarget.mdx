Get the user sleep target.

## Type

```ts
() => sleepTarget
```

## Parameters

### sleepTarget

|                       Description                     |   Type   |
| ----------------------------------------------------- | -------- |
| The user-set sleep target, default is `0`, in minutes | `number` |

## Code example

```js
const target = hmSetting.getSleepTarget()
```
