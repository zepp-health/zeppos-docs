Return the current year, month and day order.

## Type

```ts
() => result
```

## Parameters

### result: number

| Value | Format |
| --- | -------- |
| `0` | year-month-day |
| `1` | day-month-year |
| `2` | month-day-year |

## Code example

```js
const dateFormat = hmSetting.getDateFormat()
```
