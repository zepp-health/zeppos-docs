Get the weight unit set by the user.

## Type

```ts
() => weightUnit
```

## Return value

### weightUnit: number

| Value | Description |
| ----- | ----------- |
|  `0`  |   Kilogram  |
|  `1`  |     Jin     |
|  `2`  |    Pound    |
|  `3`  |    Stone    |

## Code examples

```js
const unit = hmSetting.getWeightUnit()
```
