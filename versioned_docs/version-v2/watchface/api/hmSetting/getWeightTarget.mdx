Get the user's weight target.

## Type

```ts
() => weightTarget
```

## Parameters

### weightTarget

|                       Description                 |   Type   |
| ------------------------------------------------- | -------- |
| user-set weight target, default is `0`, in kilogram | `number` |

## Code example

```js
const target = hmSetting.getWeightTarget()
```
