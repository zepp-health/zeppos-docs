Call `setScreenOff` to rest the screen.

## Type

```ts
() => result
```

## Parameters

### result

|                               Description                         |   Type   |
| ----------------------------------------------------------------- | -------- |
| The result of the operation, `0` means the setting was successful | `number` |

## Code example

```js
const result = hmSetting.setScreenOff()
```
