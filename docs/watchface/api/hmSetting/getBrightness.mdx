Returns the screen brightness of the current device.

## Type

```ts
() => result
```

## Parameters

### result

|                          Description                               |   Type   |
| ------------------------------------------------------------------ | -------- |
| The screen brightness of the current device, in the range [0, 100] | `number` |

## Code example

```js
const result = hmSetting.getBrightness()
```
