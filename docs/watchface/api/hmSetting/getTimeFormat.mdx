sidebar_label: getTimeFormat

Returns the current system-set time system.

## Type

```ts
() => result
```

## Parameters

### result: number

| Value | Format |
| --- | -------- |
| `0` | 12-hour system |
| `1` | 24-hour system |

## Code example

```js
const timeFormat = hmSetting.getTimeFormat()
```
