# DataWidget

> Start from API_LEVEL `3.6` . Please refer to [API_LEVEL](../../../../guides/framework/device/compatibility.md).

Register DataWidget, specify the lifecycle callback for the current DataWidget, etc. Each DataWidget file must call the `DataWidget()` constructor only once.

## Type

```ts
function DataWidget(option: Option): Result
```

## Parameters

### Option

| Property  | Type                                       | Required | DefaultValue | Description                                                                                                  | API_LEVEL |
| --------- | ------------------------------------------ | -------- | ------------ | ------------------------------------------------------------------------------------------------------------ | --------- |
| state     | <code>object</code>                        | N        | -            | A data object mounted on a DataWidget instance that can be used to store the state of the current DataWidget | 3.6       |
| onInit    | <code>(params?: string) =&#62; void</code> | N        | -            | It is triggered once per DataWidget and can be used to initialize the DataWidget state                       | 3.6       |
| build     | <code>(params?: string) =&#62; void</code> | N        | -            | Triggered after `onInit` execution completes, recommended for UI drawing in the `build` lifecycle            | 3.6       |
| onResume  | <code>() =&#62; void</code>                | N        | -            | Triggered when the screen focus is on this DataWidget                                                        | 3.6       |
| onPause   | <code>() =&#62; void</code>                | N        | -            | Triggered when the screen focus leaves this DataWidget                                                       | 3.6       |
| onDestroy | <code>() =&#62; void</code>                | N        | -            | The `onDestroy` lifecycle function is triggered when the DataWidget is destroyed                             | 3.6       |

### Result

| Type                 | Description         |
| -------------------- | ------------------- |
| <code>unknown</code> | DataWidget instance |

## Example

```js title="DataWidget.js"
DataWidget({
  state: {
    text: 'Hello Zepp OS',
  },
  onInit() {
    console.log('onInit')
  },
  build() {
    console.log('build')
    console.log(this.state.text)
  },
})
```
