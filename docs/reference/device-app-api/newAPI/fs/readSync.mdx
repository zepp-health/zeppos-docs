# readSync

> Start from API_LEVEL `2.0` . Please refer to [API_LEVEL](../../../../guides/framework/device/compatibility.md).

Synchronously reads the content from the file specified by the file handle into the given `ArrayBuffer`..

## Type

```ts
function readSync(option: Option): Result
```

## Parameters

### Option

| Property | Type                     | Required | DefaultValue | Description                                                              | API_LEVEL |
| -------- | ------------------------ | -------- | ------------ | ------------------------------------------------------------------------ | --------- |
| fd       | <code>number</code>      | Y        | -            | File handle, returned by the `openSync`, `openAssetsSync` and other APIs | 2.0       |
| buffer   | <code>ArrayBuffer</code> | Y        | -            | The ArrayBuffer that the data will be written to                         | 2.0       |
| options  | <code>Options</code>     | N        | -            | Other Options                                                            | 2.0       |

### Options

| Property | Type                          | Required | DefaultValue                   | Description                                                                                                                                                                           | API_LEVEL |
| -------- | ----------------------------- | -------- | ------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------- |
| offset   | <code>number</code>           | N        | <code>0</code>                 | The position in buffer to write the data to                                                                                                                                           | 2.0       |
| length   | <code>number</code>           | N        | <code>buffer.byteLength</code> | The number of bytes to read, the default is the number of bytes passed into the buffer                                                                                                | 2.0       |
| position | <code>number&#124;null</code> | N        | <code>null</code>              | Specifies the position from which to start reading from the file. If `position` is `null`, the data will be read from the current file position and the file position will be updated | 2.0       |

### Result

| Type                | Description              |
| ------------------- | ------------------------ |
| <code>number</code> | The number of bytes read |

## Example

```js
import { openSync, readSync, O_RDONLY } from '@zos/fs'

const fd = openSync({
  path: 'test.txt',
  flag: O_RDONLY,
})

const buffer = new ArrayBuffer(4)
const result = readSync({
  fd,
  buffer,
})

if (result === 0) {
  console.log('readSync success')
}
```
