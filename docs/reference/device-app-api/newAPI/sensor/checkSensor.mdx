# checkSensor

> Start from API_LEVEL `3.0` . Please refer to [API_LEVEL](../../../../guides/framework/device/compatibility.md).

Check the availability of sensors on the current device.

## Type

```ts
function checkSensor(sensor: Sensor): Result
```

## Parameters

### Sensor

| Type                | Description                                                                                                             |
| ------------------- | ----------------------------------------------------------------------------------------------------------------------- |
| <code>object</code> | Sensor, such as checking if the positioning sensor is available, pass in the `Geolocation` sensor construction function |

### Result

| Type                 | Description                                                                    |
| -------------------- | ------------------------------------------------------------------------------ |
| <code>boolean</code> | `true` - auto-brightness is set to on, `false` - auto-brightness is set to off |

## Example

```js
import { checkSensor, Geolocation } from '@zos/sensor'

const result = checkSensor(Geolocation.sensorID)
let geolocation = null

if (result) {
  geolocation = new Geolocation()
}
```
